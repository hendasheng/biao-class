魔法四季
结合“二十四节气”的自然规律，以继承传统优秀文化、传播中国文化成果为设计灵感，自主研发设计出《魔法四季》这一课程，努力实现中华传统文化在学生中的创造性转化和创新性发展。
文化发掘
“二十四节气”是中国先秦时期开始订立、汉代完全确立的用来指导农事的补充历法。它把太阳周年运动轨迹划分为24等份，每一等份为一个节气，始于立春，终于大寒，周而复始，既是历代官府颁布的时间准绳，也是指导农业生产的指南针，日常生活中人们预知冷暖雪雨的指南针，是汉族劳动人民长期经验的积累成果和智慧的结晶。
“二十四节气”于2016年11月30日入选“联合国教科文组织人类非物质文化遗产名录”。
目标意义
学生通过实践课程，锻炼其记忆力、观察能力与策略能力。同时通过设置的综合记忆、查验能力于一体的“配对”环节，促进学生大脑主管记忆、情感耐受方面神经系统的发展，锻炼其归纳整合能力。
